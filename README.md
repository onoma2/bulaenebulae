# bulaenebulae
Qubit Nebulae V2 Instruments

Everything that is not labeled unstable is stable on Nebulaer Raspberry Pi4
Controls and "docs" are inside the files for now.

##Delays
1) Kandahar to Havana - 4 stereo delays in a row with several options for cascading feedback
2) Dvedma - lo-fi feedback multitap with a filter and stereo tape emulation, absorption and stuff.
3) Perverse Engineering - I never owned an Echophne, wanted to try it.
4) Surely You Are Joking Mr. Hilbert - delay to reverb with several taps and a frequency shifter in a loop while you are waiting for the certain cinematic oscillator to arrive in the post.

Pure native Csound.

No firmware updates required. Drop it on the USB and load the instrument as usual
